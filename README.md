## Структура проекта

`client/` включает файлы клиентской части:  
- `index.html` - стартовая страница с данными о лабораторной работе  
- `about.html` - страница с персональными сведениями  

`docs/` включает файлы:  
- `markdown_examples.md` - образцы оформления элементов на Markdown  
- `latex_examples.md` - образцы математических выражений на LaTeX  

 `repo/`содержит скриншоты, демонстрирующие вид проекта в ходе его выполнения:  
- `image1_Chaykalak.png` - открытая страница index.html в браузере   
- `image2_Chaykalak.png` - работа программы в терминале  
- `image3_Chaykalak.png` - структура папок проекта  
- `image4_Chaykalak.png` - история коммитов в GitHub    

`server/` включает программный код серверной части на C#:  
- `Program.cs` - консольное приложение с интерактивным меню  

`terminal_practice/` содержит две папки:  
- `data/` включает текстовый документ `text.txt`  
- `logs/` включает документ `app.log`
