# Примеры Markdown
### Заголовки

Все заголовки разделены на уровни и пишутся со знаком "#". Чем больше количество #, тем больше уровень заголовка и тем меньше сам заголовок. Смотри [пример](/docs/markdown_examples.md) в разделе заголовки.

### Списки в Markdown

Списки помогают структурировать информацию, перечислять пункты и
облегчать восприятие текста.
Markdown поддерживает данные виды списков:
**Маркированные списки**
Создаются с помощью символов: *,+,-
- 1
- 2
- 3

**Нумерованные списки**
Создаются с помощью:
1. Первый пункт
2. Второй пункт
3. Третий пункт

**Вложенные списки**
Просто используйте отступ (таб или 4
пробела):
1. Пункт
    - Подпункт
    - Подпунк
    
### Картинка в Markdown

Изображения
Синтаксис похож на ссылки, но начинается с !:
```
![Описание изображения](путь/к/файлу.png)
```
Для локальных изображений путь указывается относительно .md файла
Пример:
![Картинка](/repo/image1_Chaykalak.png)
### Блоки кода

Markdown позволяет вставлять фрагменты кода с подсветкой синтаксиса, а также
цитаты.
Блоки кода используются со знаком `
Можно указать язык: python, js, bash, csharp и др.
```python
print("Hello, Markdown!")
```
### Пример LaTeX
- Inline-формулы    
Используются для небольших выражений внутри
строки. Оформляются в `$ ... $`. 
Пример:
Площадь круга вычисляется по формуле $S = \pi r^2$.
- Блочные формулы
Используются для больших выражений и выделяются визуально. Оформляются в `$$ ... $$`.
Пример:
$$
\int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}
$$
